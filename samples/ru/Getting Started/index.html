СКОБКИ (BRACKETS)

This is your guide!

Добро пожаловать в недавно запущенный проект Скобки (Brackets) новый редактор кода с открытым кодом для следующего поколения Интернета. Мы - большие поклонники стандартов и очень хотим создать приложения для редактирования скриптов JavaScript, HTML файлов, CSS стилей и других похожих открытых интернет технологий. Это наше скромное начало.

Скобки (Brackets) - очень молодой проект. Скобки (Brackets) - не совсем обычный редактор. Одно из главных его отличий - код самого редактора написан на JavaScript. И возможно вы еще не можете включить Скобки в свой ежедневный рацион разработчика, мы используем редактор ежедневно для его же разработки.

Несколько иноваций

Быстрое редактирование CSS стилей and JavaScript скриптов

После нажатия комбинацию клавиш Cmd/Ctrl + E во время редактирования HTML появится встроенный редактор с ссылками на все файлы стилей CSS. Изменив любой стиль CSS и нажав ESC вы вернетесь обратно к исходному файлу HTML. Можно также не закрывая CSS стили продолжать работать с HTML. Нажав ESC за пределами встроенного редактора, вы минимизируете все встроенные редакторы. Больше не нужно переключатся между разными документами!

Хотите увидеть это вживую? Наведите курсор на тэг и нажмите комбинацию клавиш Cmd/Ctrl + E. После нажатия сверху должен появится редактор CSS стилей. Справа появится список CSS стилей, относящихся к данному тэгу. Стили можно пролистать нажав Alt + Up/Down и найти необходимый стиль.

Изменения CSS стилей в браузере в живом режиме

Годами вэб разработчики делают одно и тоже: сохранить и обновить! Вы сохраняете код, затем переключаетесь в браузер и обновляете его для просмотра изменений! Больше не нужно об этом беспокоится.

Скобки (Brackets) использую живой режиме с браузером и CSS стили обновляются во время их написания! Так уже можно делать в различных приложениях к браузеру, но используя Скобки больше не нужно копировать измененный CSS стиль - он уже там, где нужно. Код бежит в браузере, на находится в редакторе.

Хотите попробовать? Установите браузер Гугл Хром (Google Chrome) и нажав на иконку молнии в верхнем правом углу или комбинацию клавиш Cmd/Ctrl + Alt + P. В живом режиме HTML документ и все относящиеся к нему стили и документы CSS можно редактировать "на лету". Цвет иконки сменится с серого на золотой когда редактор установит успешное соединение с браузером. Наведите курсор на тэг и нажмите Cmd/Ctrl + E для просмотра все правил CSS стилей. Измените ширину бордюра (border) с 1px до 10px пикселей или измените значение цвета фона (background color) с "dimgray" на "hotpink". В живом режиме Вы увидите все эти изменения сразу. Не плохо, правда ли?

На данный момент, Скобки (Brackets) поддерживает живой режим для стилей CSS. Поддержка HTML и скриптов JavaScript не за горами. На сегодняшний день мы поддерживаем браузер Google Chrome, но нам хотелось бы добавить поддержку живого режима во все широко распространенные браузеры.
