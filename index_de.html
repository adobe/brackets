Über Brackets

Brackets ist ein quelloffener Editor für Webdesign und -Entwicklung, geschaffen mit Web-Technologien wie HTML, CSS und JavaScript. Das Projekt wird von Adobe erstellt und betreut, und unter einer MIT-Lizenz zur Verfügung gestellt.

Die Ideen dahinter

Für das Web, mit dem Web
Brackets ist mit HTML, CSS und JavaScript geschaffen. Wenn Sie in Brackets programmieren können, können Sie auch an Brackets programmieren.

Offene Entwicklung funktioniert
Brackets ist ein vollständig quelloffenes Gemeinschafts-Projekt. Sie möchten die Richtung beeinflussen? Abonnieren Sie die Entwickler-Mailingliste und fangen Sie an mitzumachen.

Werkzeuge sollten Ihnen nicht im Weg sein
Anstelle Ihren Arbeitsbereich mit schwebenden Paletten, Symbolleisten und Icons zu überladen, setzt Brackets den Fokus darauf, mit „Schnell-Editoren“ direkt im Quelltext kontextsensitiv Zugriff auf Ihren Inhalt zu geben, ohne dass Sie den Code aus den Augen verlieren.

Funktioniert mit Ihrem Browser
Der Browser ist Ihre Design-Ansicht. Brackets klinkt sich direkt in den Browser ein, sodass Sie in der gleichen Umgebung designen und entwickeln, für die Ihr Code später veröffentlicht wird.

Ausprobieren

Brackets ist immer noch im Ofen und muss noch ein wenig länger backen, bevor ein voll funktionsfähiger Code-Editor entstanden ist. Die Meilenstein-Builds auf download.brackets.io stehen für Mitwirkende bereit – und für diese besondere Art von verrückten Designern und Entwicklern, die gerne das Allerneuste verwenden. Performance, Stabilität und Funktionen sind alle im Aufbau. Verwenden auf eigene Gefahr.

Wenn Sie jedoch auf dem allerneusten Stand sein möchten, werfen Sie einen Blick in das Wiki für Infos, wie Sie Brackets herunterladen und einrichten können.
